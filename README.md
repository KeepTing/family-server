### :pushpin: 项目简介： 
记录家庭生活美好瞬间、家中物品存放记录，并提供一些实用的小工具（万年历、天气、家庭记账单等）

### :pushpin: 框架支持： 

  - SpringBoot
  - Mybatis Plus
